# Zentalk Emoji Pack

The **Zentalk Emoji Pack** is a collection of smooth, scalable animated emoji designed for use in chat systems, interfaces, and digital experiences. Each emoji is lightweight, resolution-independent, and optimized for performance across devices.

## Format

- **`.tgs`** – Compressed vector animation format used for final emoji exports  
- **`.json`** – Editable animation data for preview or development use  
- **`.aep`** – Source project files for design and animation work  
- **`.png` / `.webp`** – Optional static versions for fallback support
